##  Deskripsi
Program ini menggunakan konsep Object-Oriented Programming (OOP) dengan dua kelas utama:

###  Mahasiswa 1
Menyimpan data mahasiswa dengan atribut:
- npm
- nama
- kelas

Memiliki konstruktor default, konstruktor berparameter, setter/getter, dan method tampilkanData().

### MainClass
Menjalankan menu:
- Tambah data
- Tampilkan data
- Keluar

Data disimpan menggunakan ArrayList dan input memakai Scanner.
